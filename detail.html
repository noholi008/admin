Hotel Management System

The project titled “Hotel Management System” is a hotel management software for monitoring and controlling staying of the customers. This project was developed in Netbeans 8.2 and database management was handled by MySQL. This software mainly focuses on basic operations in a hotel as well as some special features like facility of user login and facility of resetting password. New users can open a user account quite easily after signing up. A user will have the opportunity to update his/her account details, visit his/her profile, search a particular room or find room and allote the room about some specific feature. By administrative login an admin can keep track of the whole system. Admins will have the access to register user, entry customer and room details, add new rooms issued by different customers, view user profiles. Overall the key purpose of this project was to provide a friendly environment to the users as well as the stuffs of the hotel to maintain the hotel the best way possible and help reduce the human efforts. It can help the user to manage the transaction and bill payment.
Software Platform:
• The project was developed using Java Swing
• Netbeans 8.2
• MySQL server

Instagram Clone App

The project titled “Instagram clone” is an application very similar to the well known social media app Instagram. This project will be developed in Android studio and database management will be handled by firebase. This application mainly focuses on basic features of the instagram. For example: the sign up,sign in features,home activity, user profile ,commenting ,reacting to other people’s post can be done.Also stories of different people can be seen from this application. Overall the key purpose of this project is to provide a friendly environment to the users to use a lite version of instagram.
Software Platform:
• The project will be developed by Android Studio
• Kotlin
• Firebase
